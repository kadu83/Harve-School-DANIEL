# Exercícios de Pandas e EDA

Esta pasta contém exercícios de revisão de `pandas` e de análise exploratória de dados (EDA) usando DataFrames simulados. Os exemplos foram criados para rodar sem precisar baixar dados externos, o que facilita o estudo em qualquer sistema operacional.

## Como usar

1. Abra a pasta `Exercicios_Pandas_EDA` no VS Code.
2. Ative o ambiente virtual do projeto ou use um terminal com Python configurado.
3. Execute cada script usando:

```bash
python 01_criar_e_explorar_dataframes.py
python 02_eda_visualizacao.py
python 03_limpeza_e_transformacao.py
python 04_filtrar_e_selecionar.py
python 05_ordenar_e_contar.py
python 06_trabalhar_com_datas.py
python 07_valores_ausentes.py
python 08_agrupar_e_agregar.py
```

## Conteúdo

- `01_criar_e_explorar_dataframes.py` — cria DataFrames de exemplo e faz exploração básica
- `02_eda_visualizacao.py` — mostra operações de EDA e gera gráficos simples
- `03_limpeza_e_transformacao.py` — faz limpeza, tratamento e transformação de dados
- `04_filtrar_e_selecionar.py` — filtra linhas e escolhe colunas específicas
- `05_ordenar_e_contar.py` — ordena registros e conta ocorrências
- `06_trabalhar_com_datas.py` — converte datas e extrai mês/dia
- `07_valores_ausentes.py` — identifica e trata valores faltantes
- `08_agrupar_e_agregar.py` — agrupa dados e agrega valores por categoria

## Objetivos

- Revisar criação de DataFrames a partir de dicionários
- Praticar consultas, filtros, agrupamentos e ordenação
- Aprender a tratar valores ausentes, duplicados e tipos de dados
- Fazer visualizações simples para entender padrões nos dados
